# 🔧 Product Failure Analysis – Machine Failure Dataset

## 📌 Project Overview

This project focuses on analyzing machine failure data to identify the factors responsible for machine breakdowns. The analysis is based on operational parameters such as air temperature, process temperature, rotational speed, torque, and tool wear.

The objective is to understand the relationship between machine conditions and failure occurrences, helping improve reliability and reduce downtime.

---

## 🎯 Objective

- Identify machine failure causes.
- Analyze important machine parameters.
- Understand failure patterns.
- Recommend preventive measures.
- Improve machine performance and reliability.

---

## 📊 Dataset Parameters

The dataset includes the following parameters:

- Product ID
- Air Temperature (K)
- Process Temperature (K)
- Rotational Speed (rpm)
- Torque (Nm)
- Tool Wear (min)
- Failure Type
- Status

---

## ⚠️ Failure Types

### Heat Dissipation Failure (HDF)
Occurs when the machine is unable to dissipate heat effectively.

### Power Failure (PWF)
Occurs due to imbalance between rotational speed and torque.

### Tool Wear Failure (TWF)
Occurs when machine tools exceed their wear limits.

### Overstrain Failure (OSF)
Occurs when excessive load is applied to the machine.

### Random Failure (RNF)
Unexpected failure caused by random operational conditions.

---

## 🔍 Analysis

The dataset indicates that machine failures are primarily influenced by:

- High operating temperatures
- Excessive tool wear
- High torque values
- Low rotational efficiency
- Lack of preventive maintenance

---

## 🚀 Recommendations

- Perform regular maintenance.
- Monitor temperature continuously.
- Replace worn tools on time.
- Optimize rotational speed and torque.
- Implement predictive maintenance systems.

---

## 📈 Expected Outcome

- Reduced machine downtime
- Improved operational efficiency
- Better machine reliability
- Early failure detection
- Increased productivity
